## Laravel

O repositorio contém um projeto inicial Laravel e utiliza o Docker para rodar o projeto.

## Como Rodar

Com Docker e docker-compose instalado executar o comando:

```
docker-compose up
```

Acessar o Navegador na url http://localhost:8000
